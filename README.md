# Auto Gear Macro

Auto Gear Macro is a lightweight keyboard & mouse macro tool  
designed for **AFK / Automation** with an easy-to-use GUI and low resource usage.

# 🇹🇭 ภาษาไทย

## Auto Gear Macro คืออะไร
Auto Gear Macro คือโปรแกรมมาโครสำหรับคีย์บอร์ดและเมาส์  
ออกแบบมาเพื่อการ AFK และงาน Automation ที่ต้องการความเรียบง่าย  
ไม่ต้องเขียนสคริปต์ และใช้ทรัพยากรเครื่องต่ำ

เวอร์ชัน **.exe** สามารถเปิดใช้งานได้ทันที  
**ไม่ต้องติดตั้ง Python**

---

## คุณสมบัติหลัก
- กดปุ่มคีย์บอร์ดอัตโนมัติ
- กดคอมโบ เช่น `Ctrl + A`
- กดปุ่มค้าง (Hold key)
- คลิกเมาส์ซ้าย / ขวา
- กดเมาส์ค้าง
- ย้ายเมาส์ไปตำแหน่งที่บันทึกไว้
- เลือกหน่วยเวลาได้: ms / วินาที / นาที
- ทำซ้ำแบบไม่จำกัด หรือกำหนดจำนวนครั้ง
- ครบจำนวนที่ตั้งไว้แล้วหยุดอัตโนมัติ
- Export / Import มาโครเป็นไฟล์ `.json`
- System Tray
- รองรับหลายภาษา (ไทย / อังกฤษ / จีน)
- ใช้ทรัพยากรเครื่องต่ำ เหมาะสำหรับเปิดทิ้งไว้นาน

---

## วิธีใช้งานเบื้องต้น
1. เปิดไฟล์ `Auto Gear Macro.exe`
2. ตั้งปุ่ม Start / Stop (ค่าเริ่มต้น: F6)
3. กด **Add action** เพื่อเพิ่มคำสั่ง
4. ตั้งค่าเวลาและโหมด Repeat
5. กดปุ่มลัดเพื่อเริ่มหรือหยุดมาโคร

---

## Mouse Move (สำคัญ)
1. เลือก Action: **Mouse Move**
2. เลื่อนเมาส์ไปตำแหน่งที่ต้องการ
3. กด **F5** เพื่อบันทึกตำแหน่ง

---

## System Tray
- ปิดหน้าต่าง → โปรแกรมจะไปอยู่ที่ Tray
- คลิกขวาที่ไอคอน Tray เพื่อแสดง / เริ่ม / หยุด / ออก

---

## หมายเหตุ
โปรแกรมนี้เป็นเพียงเครื่องมือกดปุ่มอัตโนมัติระดับระบบปฏิบัติการ  
การนำไปใช้กับเกมหรือซอฟต์แวร์อื่น ขึ้นอยู่กับกฎของแพลตฟอร์มนั้น

# 🇬🇧 English

## What is Auto Gear Macro
Auto Gear Macro is a keyboard and mouse automation tool  
designed for AFK usage and repetitive tasks with a simple GUI.

The **EXE version** runs instantly  
**No Python installation required**

---

## Features
- Automatic key pressing
- Key combinations (e.g. `Ctrl + A`)
- Hold key actions
- Mouse left / right click
- Mouse hold
- Mouse move to saved positions
- Time units: ms / sec / min
- Continuous or fixed repeat count
- Automatically stops after reaching repeat count
- Export / Import macros (`.json`)
- System Tray support
- Multi-language UI
- Low resource usage (AFK friendly)

---

## Basic Usage
1. Run `Auto Gear Macro.exe`
2. Set Start / Stop hotkey (default: F6)
3. Click **Add action** to create macro steps
4. Configure delay and repeat mode
5. Press hotkey to start or stop

---

## Mouse Move
1. Select **Mouse Move** action
2. Move mouse to target position
3. Press **F5** to capture the position

---

## System Tray
- Closing the window sends the app to the tray
- Right-click tray icon to show / start / stop / exit

---

## Notes
This software only automates OS-level input.  
Usage with games or other software depends on their policies.

# 🇨🇳 中文

## Auto Gear Macro 是什么
Auto Gear Macro 是一款键盘和鼠标自动化工具  
适用于 AFK 和重复操作任务，界面简单，资源占用低。

EXE 版本可直接运行  
**无需安装 Python**

---

## 主要功能
- 自动按键
- 组合键（例如 `Ctrl + A`）
- 按键保持（Hold）
- 鼠标左键 / 右键点击
- 鼠标按住
- 鼠标移动到指定位置
- 时间单位：毫秒 / 秒 / 分钟
- 无限循环或指定次数循环
- 达到次数后自动停止
- 导出 / 导入宏（`.json`）
- 系统托盘支持
- 多语言界面
- 低资源占用，适合长时间运行

---

## 基本使用方法
1. 运行 `Auto Gear Macro.exe`
2. 设置开始 / 停止快捷键（默认 F6）
3. 点击 **Add action** 添加动作
4. 设置延迟和循环模式
5. 使用快捷键开始或停止

---

## 鼠标移动
1. 选择 **Mouse Move** 动作
2. 将鼠标移动到目标位置
3. 按 **F5** 保存位置

---

## 系统托盘
- 关闭窗口后程序进入托盘
- 右键托盘图标可显示 / 开始 / 停止 / 退出

---

## 注意事项
本程序仅用于系统级输入自动化  
是否可用于游戏或其他软件取决于其使用规则
